Emotion Based Music AI

GitHub Repository

Skills Gained

Python
Flask

Description

  • Developed a dynamic Music Recommendation System using Python and Flask, incorporating emotion-based song recommendations through real-time facial detection technology to match user moods with suitable tracks.
  • Created a user-friendly webpage to host the system, enabling easy access via web browsers for seamless interaction with the recommendation platform.
  • Integrated personalized singer selection features, allowing users to customize their music preferences, with the ability to add new artists dynamically.
  • Leveraged facial recognition algorithms to analyze emotions and recommend songs accordingly, providing a tailored and engaging music-listening experience.
  • Integrated YouTube playlists directly into the system, allowing for smooth playback of recommended songs from the platform, enhancing user experience and convenience.
